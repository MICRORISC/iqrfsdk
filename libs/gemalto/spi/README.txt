IQRF SPI library for Gemalto Concept Board [1] [2]
--------------------------------------------------

Introduction
------------

The library aims to help the user with a task of communication with TR [3] and DCTR [4] 
modules running either IQRF OS [5] or DPA framework [4]. 


Public Interface
----------------

See SPI_Master interface for the user's access to the TR and DCTR module.
